# TMDB Movie App
A simple iOS application that displays a list of Movies and TV Shows with infinite scrolling, search, filtering, and detail views. The app is implemented using UIKit with MVVM architecture.

## Features
- Discover Movies and TV Shows
- Search by name
- Filter by genre
- Movies and TV Shows Detail (playable Youtube trailer, Overview, Reviews from users)
